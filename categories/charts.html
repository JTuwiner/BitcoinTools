Bitbo Charts

Bitbo helps you invest in & trade Bitcoin with data, not guesses. Their dashboard provides charts, indicators, instant alerts, and in-depth monthly reports to help navigate Bitcoin with confidence.

  • Standout features: 100+ free Bitcoin charts
  • Price: Free plan available, Pro plans from $35/month
  • Mobile app: Yes

Bitbo provides extensive Bitcoin market data. We offer many charts for free users, while our paid plans start at $35/month and unlock additional features like private charts, alerts, private indicators, and real-time alerts.


TradingView

TradingView is a popular charting platform for technical analysis. It offers advanced charting tools, indicators, and customizable layouts.

  • Standout features: Ability to draw on charts
  • Price: Free plan available, Pro plans from $12.95/month
  • Mobile app: Yes

Glassnode

Glassnode is a platform for on-chain analytics. It provides detailed network data and metrics.

  • Standout features: Focus on on-chain data
  • Price: Free plan available, Pro plans from $29/month
  • Mobile app: No

TransactionFee.info

TransactionFee.info is a platform for Bitcoin network and fee analysis. It provides fee metrics and mempool data.

  • Standout features: Focus on transaction fees and data
  • Price: Free
  • Mobile app: No

Bitcoin Visuals

Bitcoin Visuals is a concise website featuring charts about the Bitcoin network.

It's less focused on traders, and more focused on the Bitcoin network.

  • Standout features: Free CSV data download
  • Price: Free
  • Mobile app: No
